## Дополненительные функции

- `#LBL1` \
  10% обучающей выборки используется для валидации.
- `#LBL3` \
  Веса модели автоматически сохраняются в конце эпохи, если качество на валидации превысило предыдущие результаты.
- `#LBL5` \
  После каждой эпохи выводятся loss и accuracy на тренировочной и валидационной выборках.
- `#LBL6` \
  Строятся графики loss и accuracy на тренировчной и валидационной выборках.
- `#LBL11` \
  Для тренировочного набора используются аугментации: вертикальное и горизонтальное отражения, преобразование цветов (ColorJitter), кадрирование. \
  Аугментации были добавлены, чтобы побороть переобучение, которое наблюдалось без них. \
  Так как основным фактором при определении класса является текстура ткани, то были выбраны аугментации, не влияющие на саму текстуру.
